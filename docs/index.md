???+ tip

    想快速构建一个类似的知识库？ Material for MkDocs 主题是个不错的选择。仅需三步即可完成最小配置 (1)
    { .annotate }
    
    1. 插件系统基于 python，配置简单，非常适合前端 0 基础
    
    ```bash title="通过 pip 安装 Material 主题"
    pip install mkdocs-material
    ```

    ```bash title="创建一个新的 MkDocs 项目"
    mkdocs new .
    ```

    ```bash title="启动开发服务器"
    mkdocs serve
    ```
